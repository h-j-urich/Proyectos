Materia: Matemáticas - Semana de Integración 1.

Proyecto seleccionado:  2 - Conversión de Números 

Objetivo: Desarrollar un programa que convierta números decimales a binarios y, de forma opcional, 
también de binario a decimal. Validar la entrada y mostrar mensajes de error ante datos incorrectos. 

Requisitos:
- Python 3.11.9
- No requiere librerías externas

Instalación:
- Clonar el repositorio o descargar el archivo "Integracion_1.py".
- Abrir una terminal y ejecutar: "python Integracion_1.py".

Uso:
El programa solicita al usuario que elija una conversión por medio de un menu de opciones.-

Resumen del script:
- Validación de entrada para evitar errores con datos incorrectos.
- Implementación de la conversión en ambos sentidos (decimal a binario y binario a decimal).
- Mensajes de error en caso de entradas no válidas.
- Uso de funciones para mejorar la estructura del código (modulos).
